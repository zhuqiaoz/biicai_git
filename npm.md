# npm

1. 安装模块(局部安装)
```
    npm install mock
```
2. 全局安装
```
    npm install mock -g
```
3. 查看已安装的模块
```
    npm list(npm ls)
```
4. 卸载模块
```
    npm uninstall mock
```
5. 清除缓存
```
    npm cache clean
```
6. 查看版本号
```
    npm info angular
```
7. 更新版本
```
    npm updata angular
```
8. 安装特定版本号
```
    npm install angular@2.0.0
```




# package
* dependencies //package的应用依赖模块，即别人要使用这个package，至少需要安装哪些东东。应用依赖模块会安装到当前模块的node_modules目录下。
* devDependencies //package的开发依赖模块，即别人要在这个package上进行开发
