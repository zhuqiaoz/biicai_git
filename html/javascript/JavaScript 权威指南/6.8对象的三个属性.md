# 原型(prototype) 类(class) 可扩展性(extensible attribute)

* 要检测一个对象是否是另外一个对象的原型 isPrototypeOf()

```
var a = {a:1}
var o = Object.create(a);

console.log(a.isPrototypeOf(o))
Object.prototype.isPrototypeOf(a)
```
