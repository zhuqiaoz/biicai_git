属性赋值失败：
* 对象的属性是只读的。不能给只读的属性赋值。（Object.defineProperty()）
* 对象的属性是继承来的。只读。不能通过同名的属性来覆盖只读的继承属性。
* 对象不存在查询的属性，也不是集成的属性。


* delete：
delete 运算符只能删除本身的属性，不能删除继承的熟悉（要删除继承的属性，必须从定义这个属性的原型删除，而却这回影响所有继承这个原型的对象）
```
var a = {b : {c:1}}
p = a.b
delete a.b
alert (p.c)
```
