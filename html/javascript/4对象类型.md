+ 复杂类型(Object)

### 检查类型
* typeof  // string、number、boolean、Object、undefined和Object
* instanceof  

> 虽然在检测基本数据类型时 typeof 是非常得力的助手，但在检测引用类型的值时，这个操作符的用处不大。通常，我们并不是想知道某个值是对象，而是想知道它是什么类型的对象
